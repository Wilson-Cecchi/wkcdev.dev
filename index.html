01 Desenvolvimento de Software

Programação em C com foco em aplicações de terminal, lógica de jogos, sistemas de pontuação, manipulação de arquivos e menus interativos.

→
  • Linguagem principal: C (C99/C11)
  • Estruturas de dados: vetores, structs, ponteiros
  • Manipulação de arquivos: leitura/escrita com FILE*
  • Interface terminal: ANSI escape codes, menus interativos
  • Lógica de jogos: movimentação, colisão, ranking, timer
02 Configuração Linux

Instalação e configuração de distribuições Linux, ajuste de drivers, serviços e otimização do sistema para programação e uso geral.

→
  • Distribuições: Ubuntu, Zorin OS, ZimaOS
  • Gerenciamento de serviços: systemd, journalctl
  • Drivers: NVIDIA legacy, Intel i915, DKMS
  • Otimização: throttling, power management
  • Ambientes: GNOME, configuração de boot e GRUB
03 Infraestrutura Básica

Configuração de servidores pessoais, máquinas virtuais e compartilhamento de arquivos em rede com Samba e afins.

→
  • Servidores: ZimaOS, Docker, Docker Compose
  • Rede: Tailscale VPN, SSH (ed25519), fail2ban
  • Compartilhamento: Samba, SCP, Rsync
  • Monitoramento: Uptime Kuma, AdGuard Home
  • Backup: Restic + SFTP com retenção automática
04 Projetos & Prototipagem
